Objektum orientált programozás

Ez a tárgy megismerteti a hallgatókkal az objektum orientált programozás alapelveit, valamint a Java programozási nyelvet.

Tematika

  • Java nyelv alapelemei
  • Java nyelv utasításai
  • OOP fogalma, alapelvei
  • Osztálydefiníció:referencia,hozzáférés,konstruktor,destruktor
  • Függvénytúlterhelés,paraméter átadás,osztályszint,példányszint
  • Inicializáló blokk,konstansok,'final' kulcsszó
  • Öröklődés, polimorfizmus, overriding,referenciák statikus és dinamikus tipusa
  • Absztrakt metódusok,absztrakt osztályok
  • Interfészek,csomagok,névterek
  • Kivételkezelés,Beágyazott osztályok,interfész tagok, tömb és enum típus
  • Osztályok közti kapcsolatok,fájlkezelés
  • Generikus programok Java-ban, java.lang csomag osztályai
